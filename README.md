Objetivos
------------
Colocar as questões envoldendo lógica de programação em C.\
Apresenta um guia de estudos para a linguagem.

-----------

3- Vetores e Matrizes \

5- Struct , Unions e Enums \
6- Ponteiro e Alocação dinamica
